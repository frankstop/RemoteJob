# RemoteJob Design Context

## Register

Product UI.

## Scene

Remote job seeker checking new postings during a morning application block on a laptop in a bright room. The interface should be calm, fast to scan, and useful for repeated daily work.

## Theme

Light product UI. Tinted neutral surfaces with one green-blue accent.

## Tokens

- Background: `oklch(0.965 0.008 178)`.
- Surface: `oklch(0.992 0.005 178)`.
- Accent: `oklch(0.54 0.13 181)`.
- Text: `oklch(0.19 0.018 182)`.
- Radius: `8px`.
- Font: system sans.

## Component Rules

- Use rows and panels for work surfaces.
- Keep cards shallow, never nested.
- Standard buttons, selects, inputs, toggles.
- Icon buttons use consistent lucide outline icons.
- Status chips use semantic tints.
